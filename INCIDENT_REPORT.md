- Added `logging` import and configured a module-level logger for tenacity to emit warnings before each sleep and info after each attempt.
- Introduced `is_rate_limit_error(exception)` predicate that returns `True` only when the exception is an `HTTPError` with HTTP status code 429, ensuring other errors (500, connection errors, etc.) are NOT retried.
- Decorated `sync_data()` with `@retry` using:

  - `stop=stop_after_attempt(6)` — gives up after 6 total attempts.
  - `reraise=True` — re-raises the original `HTTPError` instead of wrapping it in `RetryError`.
- Original function signature and docstring are fully preserved.
