## Tech Stack

| Component       | Technology                                    |
|-----------------|-----------------------------------------------|
| Agentic Framework | **LangGraph** (StateGraph with typed state) |
| LLM             | **Claude Sonnet** via `langchain-anthropic`   |
| Mock Server     | **FastAPI** + Uvicorn                         |
| Retry Library   | **tenacity** (injected by the coder agent)    |
| GitHub API      | **PyGithub**                                  |
| Notifications   | **Slack Incoming Webhooks** via `httpx`        |
| Environment     | **python-dotenv**                             |

## LangGraph Pipeline Architecture

```
┌─────────────┐     ┌──────────────┐     ┌───────────┐     ┌──────────┐
│   classify   │────▶│   codegen    │────▶│  open_pr  │────▶│  notify  │──▶ END
│  (Router)    │     │  (Coder)     │     │ (GitHub)  │     │ (Slack)  │
└─────────────┘     └──────────────┘     └───────────┘     └──────────┘
       │
       │ fault_type == "unknown"
       ▼
      END (skip)
```

**State Flow:**
1. `classify` — Sends crash log to Claude, returns fault classification JSON.
2. Conditional edge — If `fault_type == "unknown"`, pipeline stops.
3. `codegen` — Sends original source + classification to Claude, returns fixed code + test.
4. `open_pr` — Uses PyGithub to create branch, commit 3 files, open PR.
5. `notify` — Sends structured Slack message with triage details + PR link.

## Environment Variables

| Variable             | Description                                      |
|----------------------|--------------------------------------------------|
| `ANTHROPIC_API_KEY`  | Anthropic API key for Claude                     |
| `GITHUB_TOKEN`       | GitHub Personal Access Token (needs `repo` scope)|
| `GITHUB_REPO`        | Target repo in `owner/name` format               |
| `SLACK_WEBHOOK_URL`  | Slack Incoming Webhook URL                       |
| `MOCK_SERVER_URL`    | Mock server base URL (default: http://localhost:8429) |

## Setup

```bash
# 1. Clone & install
git clone https://github.com/tanmeipheng/push-to-prod-mediacorp.git
cd push-to-prod-mediacorp
uv sync

# 2. Configure environment
cp .env.example .env
# Edit .env with your actual keys

# 3. Run the demo
./demo.sh
```

## Running Individual Components

```bash
# Start mock server (Terminal 1)
uv run python mock_server/server.py

# Run the crash (Terminal 2) — just to see the crash
uv run python vulnerable_app/integration.py

# Run only the crash capture
uv run python crash_runner/run_and_capture.py

# Run the full pipeline (requires mock server running)
uv run python main.py

# Run with a pre-captured log file
uv run python main.py --log path/to/crash.log
```
